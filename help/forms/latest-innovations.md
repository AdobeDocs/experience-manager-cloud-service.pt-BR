# Principais inovações do Adobe Experience Manager Forms {#latest-innovations}

Algumas das principais inovações no Adobe Experience Manager Forms incluem:

| Inovação | Detalhes |
|---|---|
| Forms adaptável headless | Criar e gerenciar [Forms adaptável headless](https://experienceleague.adobe.com/docs/experience-manager-headless-adaptive-forms/using/overview.html) na plataforma Adobe Experience Manager. Permita que seus desenvolvedores criem, publiquem e gerenciem formulários interativos que podem ser acessados e interagidos com por meio de APIs, em vez de por meio de uma interface gráfica tradicional. <br/> <br/> Esses formulários são projetados para serem enviados sem a necessidade de uma interface de forma HTML tradicional. Em outras palavras, elas permitem enviar dados de formulário de forma programática por meio de uma API ou código de back-end com e sem elementos de formulário visíveis no front-end. <br/> ![](https://experienceleagueadobe.com/docs/experience-manager-headless-adaptive-forms/assets/how-headless-adaprive-forms-work.png?)<br/> Formulários headless são úteis em vários cenários, como ao criar aplicativos de página única, aplicativos web progressivos ou aplicativos móveis, onde uma interface de formulário HTML tradicional pode não ser necessária ou prática. Ao permitir que os desenvolvedores enviem dados de formulário diretamente por meio de APIs ou código de back-end, os formulários sem periféricos ajudam a simplificar os fluxos de trabalho e a melhorar o desempenho geral dos aplicativos Web. |
| Componentes principais  | O [Componentes principais adaptáveis do Forms](https://experienceleague.adobe.com/docs/experience-manager-core-components/using/adaptive-forms/introduction.html#features) são um conjunto de 24 componentes de código aberto compatíveis com BEM, criados na base dos Componentes principais do WCM Adobe Experience Manager. Eles foram especificamente projetados para serem usados na criação dos Formulários adaptáveis, que são formulários que se adaptam ao dispositivo, navegador e tamanho da tela do usuário. <br/> <br/> Esses componentes podem ser usados para criar experiências excepcionais de captura e registro de dados, fornecendo uma grande variedade de opções de campos de formulário, incluindo campos de texto, caixas de seleção, menus suspensos e muito mais. Eles também incluem recursos como validação, lógica condicional e design responsivo, que podem ser usados para criar formulários fáceis de usar e intuitivos. <br/> ![](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/assets/sample-core-components-based-adaptive-form.png?)<br/>  Além disso, como esses componentes são de código aberto, os desenvolvedores têm a capacidade de personalizar e estender facilmente os componentes para atender às necessidades específicas de sua organização. Além disso, esses componentes são criados com base na metodologia BEM, que garante a capacidade de expansão e manutenção. |
| Conector Microsoft PowerAutomate | O AEM Forms Power Automate Connector permite integrar o Adobe Experience Manager (AEM) Forms com o Microsoft Power Automate (anteriormente conhecido como Fluxo do Microsoft). O Power Automate é um serviço baseado em nuvem que permite criar fluxos de trabalho automatizados entre diferentes aplicativos e serviços.  <br/> <br/> Com AEM conector do Form Power Automate, você pode criar fluxos de trabalho que são acionados automaticamente com base no envio de um formulário adaptável. Por exemplo, é possível criar um fluxo de trabalho que envie automaticamente uma notificação por email para uma pessoa específica quando um usuário envia um formulário ou cria uma tarefa no Microsoft Planner quando um usuário conclui um formulário.  <br/> ![](https://powerusers.microsoft.com/t5/image/serverpage/image-id/182924i17C4BEA1C045D731/image-size/large/is-moderation-mode/true?v=1.0&amp;px=999) <br/> O AEM Forms Power Automate Connector é uma ferramenta poderosa que permite automatizar e integrar seu Adaptive Forms com outros aplicativos e serviços que se conectam ao Microsoft Power Automate, permitindo que você trabalhe com uma grande variedade de ferramentas. Você pode criar fluxos de trabalho adaptados às suas necessidades específicas, com a capacidade de adicionar ações, condições e acionadores personalizados. Além disso, o Power Automate fornece análises e relatórios detalhados, permitindo que você monitore e otimize seus fluxos de trabalho ao longo do tempo. |
| Conectores de armazenamento Microsoft | Conectores de armazenamento AEM Forms Microsoft para <a href="https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/forms/adaptive-forms-authoring/authoring-adaptive-forms-foundation-components/configure-submit-actions-and-metadata-submission/configuring-submit-actions.html#submit-to-sharedrive">OneDrive</a>, <a href="https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/forms/adaptive-forms-authoring/authoring-adaptive-forms-foundation-components/configure-submit-actions-and-metadata-submission/configuring-submit-actions.html?#submit-to-sharedrive"> SharePoint, </a> e <a href="https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/forms/adaptive-forms-authoring/authoring-adaptive-forms-foundation-components/configure-submit-actions-and-metadata-submission/configuring-submit-actions.html?#submit-to-azure-blob-storage"> Armazenamento Azure Blob </a> são conectores que permitem integrar o Adobe Experience Manager (AEM) Forms com o Microsoft OneDrive e o SharePoint. Com esse conector, você pode fazer upload de arquivos de dados e anexos para o OneDrive e o SharePoint diretamente do Adaptive Forms. <br/> ![](/help/forms/assets/onedrive-and-sharepoint.jpg) <br/>O OneDrive e o SharePoint podem se integrar a outros aplicativos de negócios, como sistemas de CRM, software de contabilidade e ferramentas de gerenciamento de projetos. Isso permite que você simplifique seus processos de negócios, reduza a entrada manual de dados e melhore a eficiência geral. |
| Interface do usuário do assistente | A interface do Assistente adaptável Forms é uma ferramenta poderosa para criar formulários adaptáveis de forma rápida e fácil. Suas opções de personalização e interface amigáveis o tornam acessível a todos os usuários, independentemente do nível de conhecimento técnico. <br/> <br/> A interface do usuário do assistente simplifica o processo de criação de formulários adaptáveis, orientando os usuários passo a passo pelo processo de criação de formulários. A interface do usuário do assistente é dividida em várias guias, com cada guia fornecendo claramente as opções para configurar um formulário adaptável. Um autor de formulário avança pelas guias de forma linear para selecionar opções como modelos, enviar ações e fontes de dados para componentes do formulário. <br/> ![](/help/release-notes/assets/wizard.png) <br/>A interface do assistente simplifica o processo de descoberta de todas as opções essenciais para um formulário adaptável e facilita a criação do formulário, mesmo para usuários que não estão familiarizados com a tecnologia. |
| Adobe Analytics com Experience Cloud Setup Automation para Forms  [!BADGE Em breve]{type=Informative tooltip="O Adobe Analytics com Experience Cloud Setup Automation for Forms será lançado em breve"} | A análise de formulários pode fornecer informações valiosas sobre o desempenho dos formulários, medindo o envolvimento do usuário, otimizando as taxas de conversão, monitorando o desempenho do formulário e melhorando a experiência do usuário.  Ao rastrear o comportamento do usuário e o feedback, o Analytics pode identificar áreas do formulário que estão causando frustração ou confusão, orientando melhorias no design e na funcionalidade do formulário. <br/> <br/> Ao usar o recurso Você pode ativar o Adobe Analytics com o Experience Cloud Setup Automation com alguns botões. Ela permite conectar o AEM Forms as a Cloud Service com tags Experience Platform e Adobe Analytics para capturar e rastrear métricas de desempenho de seus formulários publicados. <br/> <br/> ![](/help/forms/assets/forms-analytics-report.png) <br/><br/> O Forms as a Cloud Service fornece um relatório OOTB da Adobe Analytics. Ele ajuda você a entender facilmente o desempenho de seus formulários. As métricas no nível do formulário fornecem um insight sobre o desempenho do formulário em vários KPIs (indicadores chave de desempenho), como representações, visitantes, envios, tempo médio de preenchimento. <br/> <br/> Ela também fornece detalhes sobre o número médio de vezes que os usuários acessaram a ajuda no contexto dos campos no painel para ajudar a determinar os campos que fazem com que os usuários parem e procurem informações antes de fornecer informações. Você pode simplificar ainda mais esses campos ou ajudar o conteúdo a melhorar as conversões. |
| Composição da próxima geração | Com a próxima composição de geração, os funcionários em toda a empresa podem criar formulários nas ferramentas preferidas, como Microsoft Excel e Google Sheets, sem a necessidade de treinamento extenso sobre novos workflows da interface do usuário. <br/> <br/> Além disso, a Composição da próxima geração foi projetada para carregar formulários com velocidade incomparável, atingindo uma pontuação do Google Lighthouse quase perfeita nas páginas ativas, garantindo o melhor desempenho e a experiência do usuário.<br/> <br/> ![](/help/forms/assets/web-vitals.jpeg) Além disso, a implantação de formulários nunca foi tão fácil ou rápida, graças às ferramentas simples fornecidas pela próxima geração de Composição. Com o tempo mínimo de desenvolvimento necessário, você pode simplificar facilmente o caminho para entrar em funcionamento e fornecer experiências excepcionais de formulário aos seus clientes. |
