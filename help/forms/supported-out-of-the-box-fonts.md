## Fontes prontas para uso (OOTB) com suporte {#ootb-supported-fonts}

As listas de fontes OOTB suportadas são:

* AdobeArabic
* AdobeFanHeitiStd
* AdobeGothicStd
* AdobeHebrew
* AdobeHeitiStd
* AdobeThai
* LetterGothicStd
* MinionPro
* MyriadPro
* SymbolStd
* AdobeMingStd
* AdobeMyungjoStd
* AdobePiStd.otf
* AdobeSongStd
* CourierStd
* KozGoPr6N
* KozGoProVI
* eurosign.ttf


Além das fontes mencionadas acima, é possível usar fontes personalizadas para dar aos PDFs uma aparência específica. Para habilitar fontes personalizadas, você [adiciona fontes personalizadas ao seu ambiente Forms as a Cloud Service](/help/forms/use-custom-fonts.md) com uma licença válida.
