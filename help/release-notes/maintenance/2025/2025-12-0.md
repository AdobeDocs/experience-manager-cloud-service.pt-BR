# Notas da versão de manutenção {#maintenance-release-notes}

A seção a seguir descreve as notas de versão técnicas para a versão de manutenção do Experience Manager as a Cloud Service associadas à ativação de recursos 2025.12.0.

## Versão 23482 {#23482}

Veja abaixo um resumo das melhorias contínuas da versão de manutenção 23482, lançada publicamente em quinta-feira, 3 de dezembro de 2025. A versão de manutenção anterior era 23385.

A ativação de recursos do 2025.12.0 fornece o conjunto completo de recursos para esta versão de manutenção. Consulte o [Roteiro de versões do Experience Manager](https://experienceleague.adobe.com/pt-br/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap) para obter mais informações.

### Aprimoramentos {#enhancements-23482}

* ASSETS-49770: Adicionar notificações de quarentena para resultados de verificação de malware.
* ASSETS-54079: aplicar o formulário de metadados personalizado à pasta de quarentena.
* ASSETS-54083: Criar mecanismo de limpeza de quarentena programado.
* ASSETS-54278: remova a propriedade `dam:avScanTime` dos ativos.
* ASSETS-57284: Restrinja os uploads de arquivo à pasta Quarantine (desabilite arrastar e soltar).
* ASSETS-57428: ocultar a pasta de quarentena na interface do usuário do Assets View.
* ASSETS-57626: melhore o comportamento de repetição para trabalhos de ativos assíncronos.
* ASSETS-57879: adicionar opção de mesclagem para trabalhos assíncronos de movimentação/cópia de ativos.
* ASSETS-58099: adicione configuração para desativar tags inteligentes aprimoradas para todo o ambiente.
* ASSETS-58136: Implementar feedback de paginação na OpenAPI de pesquisa.
* ASSETS-59402: Adicionar pontos de extremidade de trabalho assíncrono para a API de exclusão de pasta: exportar pacotes para região interna.
* ASSETS-59966: Renomeie o grupo de Administradores de Malware para Administradores de Quarentena.
* ASSETS-60166: Use VideoViewer.js em vez de URLs baseados em iframe.
* GRANITE-61378: Ferramenta de depuração de permissões - API ListPrincipals.
* GRANITE-63235: consulta para identificar Sites usando a propriedade `cq:conf`, suporte à detecção de páginas/versões antigas.
* SITES-30452: API de conteúdo com ASO - Sugestões de título e descrição, suporte a XWalk, operações de patch JSON, Associação principal de serviço IMS.

### Problemas corrigidos {#fixed-issues-23482}

* ASSETS-57430: Corrigir carregamento da Exibição do Assets ignorando o pré-processamento: exportar o pacote `repoapi.preprocessing`, atualizar RAPI para o mais recente.
* ASSETS-58190: reduza o total de suposições desnecessariamente altas na interface do usuário de coleções.
* ASSETS-58866: título/descrição/ID do ativo de correção retornado nas respostas OpenAPI.
* ASSETS-58868: corrija a paginação ao classificar campos ausentes em ativos.
* ASSETS-58920: corrija a importação de ativos em massa ignorando o pré-processamento.
* ASSETS-59168: Corrija a hora de início/término da verificação mostrando o fuso horário incorreto.
* ASSETS-59702: corrija a solicitação de eventos quando o status do ativo for definido como Sem status.
* ASSETS-59830: Os trabalhos assíncronos da fila novamente são interrompidos durante o encerramento do pod.
* ASSETS-49757: correções para eventos de verificação de Detecção de Malware.
* GRANITE-61019: correção `gcMonitor` não notificada na primeira execução após a reinicialização do AEM.
* GRANITE-62717: corrija o `JSafe` tratamento de senha com caracteres não ASCII.
* SITES-34331: Corrigir o tempo limite 503 ao carregar a Sobreposição de implantação para usuários não administradores (`cqLiveSyncCancelled index`).

Nenhum.
