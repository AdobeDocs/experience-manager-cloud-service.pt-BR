# Notas da versão de manutenção {#maintenance-release-notes}

A seção a seguir descreve as notas de versão técnicas para a versão de manutenção atual do Experience Manager as a Cloud Service.

## Versão 11983 {#release-12142}

Veja abaixo um resumo das melhorias contínuas da versão de manutenção 12142, lançada publicamente em 31 de maio de 2023. Esta versão de manutenção é uma atualização da versão de manutenção anterior, a 11983.

A ativação de recursos desta versão de manutenção fornecerá o conjunto completo de recursos. Consulte as [notas de versão atuais](/help/release-notes/release-notes-cloud/release-notes-current.md) para obter detalhes completos.

### Aprimoramentos {#enhancements-12142}

Nenhum.

### Problemas conhecidos {#known-issues-12142}

Nenhum.

### Problemas corrigidos {#fixed-issues-12142}

- SITES-13636 - Correção de um problema em que as dependências clientlib da imagem e do site do carrossel clientlib estavam ausentes.
- SITES-13475 - Corrigido um problema de quebra de compatibilidade com versões anteriores do PanelContainer.js na versão 2.22.10 dos Componentes principais.
