# Notas de versão de visualização do Universal Editor {#preview}

Estas são as notas de versão da **versão de visualização** do Editor Universal. Estes recursos estão disponíveis atualmente no **ambiente de visualização** do Editor Universal. Esses recursos estão programados para serem lançados para disponibilização geral em 21 de janeiro de 2026.

Estas notas de versão do **preview** são fornecidas como conveniência, para que você saiba quais alterações do Universal Editor estão por vir e possa testá-las [alternando para o ambiente de visualização.](/help/sites-cloud/authoring/universal-editor/navigation.md#user-properties)

>[!TIP]
>
>Para as **notas de versão atuais** do Universal Editor, consulte as [Notas de Versão do Universal Editor.](/help/release-notes/universal-editor/current.md)

>[!NOTE]
>
>O conteúdo da versão real, bem como a data de lançamento, estão sujeitos a alterações.

## Novos recursos futuros {#what-is-new}

* Os blocos de texto na Árvore de conteúdo mostram visualizações de texto curtas.
* Os campos de texto sem formatação são compatíveis com áreas de texto de várias linhas.
* As extensões podem usar a API da Notificação do sistema para mostrar notificações no produto.

## Aprimoramentos futuros {#other-improvements}

* O espaço em branco à esquerda no editor de rich text é limpo corretamente, evitando peculiaridades de layout e formatação.
* Pressionar Enter corretamente aciona a edição em linha novamente, restaurando o comportamento de edição esperado.
* Vários problemas de acessibilidade do navegador foram resolvidos.
* A formatação de atalho de teclado foi unificada, e os ícones na paleta de comandos foram ajustados para oferecer uma experiência mais clara e consistente.
