# Notas de versão de visualização do Universal Editor {#preview}

Estas são as notas de versão da **versão de visualização** do Editor Universal. Estes recursos estão disponíveis atualmente no **ambiente de visualização** do Editor Universal. Esses recursos estão programados para serem lançados para disponibilização geral em 12 de fevereiro de 2026.
